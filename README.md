Das aktuelle tablet-Frontend für mein fhem-Hausautomationssystem, installiert auf einem Raspberry Pi 3 Model B Rev 1.2.

Überwacht und schaltet folgende Komponenten:
- Homematic (div. Steckdosen, Schalter, Schaltkasten)
- Haus-Stromzähler Zweirichtungszähler DZG DWS7612.2V über einen tasmota Lesekopf Pro 3
- Solarthermie und Heizung (Pelletofen Montegrappa Caminetti Aktiv 15) über einen Cosmo Multi Regler, Anbindung per vbus an einen Resol LAN-Adapter
- Wetter-Display (Daten von Proplanta)
- den Raspberry Pi selbst ;-)
